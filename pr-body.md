## Resumo

Implementa typing indicators para proporcionar experiência similar ao WhatsApp Web:

- **Typing indicator recebido**: Exibe "digitando..." quando o cliente está escrevendo
- **Typing indicator enviado**: Envia indicador de "escrevendo" para o cliente quando a IA está processando a resposta
- **Reações**: Funcionalidade já existia e foi conferida (botão no hover das mensagens)
