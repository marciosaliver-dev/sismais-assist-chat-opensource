Política de Privacidade

Sismais Tecnologia — Plataforma GMS (Gestão Mais Simples)

Última atualização: 1 de abril de 2026

A Sismais Tecnologia LTDA ("nós", "nosso" ou "Sismais") valoriza a privacidade e a proteção dos dados pessoais dos seus usuários. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos, compartilhamos e protegemos as informações pessoais obtidas por meio da plataforma GMS — Gestão Mais Simples, incluindo nossos canais de atendimento via WhatsApp, site, aplicativos e integrações com serviços de terceiros.

Ao utilizar nossos serviços, você concorda com as práticas descritas nesta política, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018) e com as políticas da Meta Platforms, Inc. para uso da API do WhatsApp Business.

1 Dados que Coletamos

1.1 Dados fornecidos diretamente por você

  • Dados de identificação: nome, sobrenome, CPF/CNPJ, razão social
  • Dados de contato: número de telefone (WhatsApp), e-mail, endereço
  • Dados de atendimento: mensagens de texto, áudios, imagens e documentos enviados via WhatsApp ou outros canais de suporte
  • Dados contratuais: informações sobre contratos, licenças de software e planos contratados
  • Dados de feedback: avaliações de atendimento, pesquisas de satisfação

1.2 Dados coletados automaticamente

  • Dados de uso: páginas acessadas, funcionalidades utilizadas, horários de acesso, duração das sessões
  • Dados do dispositivo: tipo de navegador, sistema operacional, resolução de tela, endereço IP
  • Dados de WhatsApp: número de telefone, nome de perfil, foto de perfil (quando disponível publicamente), status de leitura de mensagens
  • Cookies e tecnologias similares: identificadores de sessão, preferências de interface

1.3 Dados obtidos de fontes de terceiros

  • Meta / WhatsApp Business API: dados de perfil e mensagens recebidas via API oficial do WhatsApp
  • Sistemas integrados (Sismais GL / Sismais Admin): dados cadastrais de clientes, contratos ativos, status de licenças

Sobre o uso de Inteligência Artificial: Utilizamos agentes de IA para processar mensagens e fornecer respostas automatizadas. Nenhuma decisão automatizada com efeitos legais é tomada sem supervisão humana.

3 Compartilhamento de Dados

Seus dados pessoais podem ser compartilhados com:

Destinatário Finalidade Dados compartilhados
Meta Platforms (WhatsApp) Envio e recebimento de mensagens via API oficial Número de telefone, conteúdo de mensagens
Provedores de IA (OpenRouter, OpenAI) Processamento de linguagem natural e geração de embeddings Conteúdo textual de mensagens (anonimizado quando possível)

Não vendemos, alugamos ou comercializamos seus dados pessoais com terceiros para fins de marketing.

4 Armazenamento e Segurança

  • Infraestrutura: Os dados são armazenados em servidores seguros da Supabase (PostgreSQL), com criptografia em trânsito (TLS/SSL) e em repouso
  • Controle de acesso: Políticas de Row Level Security (RLS) garantem que cada usuário acessa apenas seus próprios dados
  • Autenticação: Acesso à plataforma protegido por autenticação segura com tokens JWT
  • Retenção: Dados são mantidos pelo período necessário para cumprir as finalidades descritas nesta política ou conforme exigido por lei
  • Backups: Realizados periodicamente com criptografia
Importante: Embora adotemos medidas de segurança compatíveis com os padrões da indústria, nenhum sistema é 100% infalível. Em caso de incidente de segurança, notificaremos os titulares afetados e a ANPD conforme exigido pela LGPD.

5 Seus Direitos (LGPD)

Como titular de dados pessoais, você tem direito a:

  • Confirmação e acesso: Saber se tratamos seus dados e acessar uma cópia
  • Correção: Solicitar a correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação: Solicitar o tratamento adequado de dados desnecessários ou excessivos
  • Portabilidade: Solicitar a transferência dos seus dados a outro fornecedor
  • Eliminação: Solicitar a exclusão dos dados pessoais tratados com base no seu consentimento
  • Informação sobre compartilhamento: Saber com quais entidades seus dados foram compartilhados
  • Revogação do consentimento: Retirar seu consentimento a qualquer momento
  • Oposição: Opor-se ao tratamento realizado com base em legítimo interesse

7 Cookies e Tecnologias de Rastreamento

Utilizamos cookies estritamente necessários para o funcionamento da plataforma:

  • Cookies de sessão: Para manter sua autenticação ativa
  • Cookies de preferência: Para armazenar configurações de interface (tema, idioma)

Não utilizamos cookies de rastreamento para fins publicitários de terceiros.

8 Uso da API do WhatsApp Business

Nosso serviço de atendimento via WhatsApp opera por meio da API oficial do WhatsApp Business fornecida pela Meta Platforms, Inc. Ao nos enviar uma mensagem via WhatsApp, você concorda que:

  • Seu número de telefone e conteúdo das mensagens serão processados pela plataforma Sismais para fins de atendimento
  • Mensagens poderão ser processadas por agentes de Inteligência Artificial antes de serem respondidas
  • Dados são transmitidos de forma criptografada entre o WhatsApp e nossa plataforma
  • Você pode solicitar a exclusão do seu histórico de conversas a qualquer momento pelo canal de privacidade

Para mais informações sobre como a Meta trata seus dados, consulte a Política de Privacidade do WhatsApp.

9 Dados de Menores de Idade

Nossos serviços são destinados a empresas e profissionais (B2B). Não coletamos intencionalmente dados de menores de 18 anos. Caso identifiquemos dados de menores, estes serão eliminados imediatamente.

10 Transferência Internacional de Dados

Alguns dos nossos provedores de serviço (como Supabase, OpenAI e Meta) podem armazenar ou processar dados fora do Brasil. Nestes casos, garantimos que a transferência ocorre em conformidade com a LGPD, mediante cláusulas contratuais padrão ou para países com nível adequado de proteção de dados reconhecido pela ANPD.

11 Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Alterações significativas serão comunicadas por meio da plataforma ou pelos nossos canais de atendimento. A data da última atualização está indicada no topo deste documento.

Recomendamos que você revise esta política periodicamente para estar ciente de quaisquer mudanças.
